## interactive-maps-drivers
An interactive map to display the number of drivers. Map is generated using the Google Maps API. 

### to run the frontend
`npm run client`

### to run server
`npm run server`

### to run concurrently
`npm run dev`

- Click on the compass on the top right corner of the map
- Allow location detection on your browser
- It will pan the map the area where you're located at
